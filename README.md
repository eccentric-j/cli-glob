# cli-glob [![Travis][build-bage]][build]
A glob cli to find files quickly including reading from stdin so it can be piped to other unix apps.

## Installation
Using npm:
`sudo npm install cli-glob`

## Usage
`glob 'src/*.js'`

## Output
```
src/cli.js
src/index.js
```
